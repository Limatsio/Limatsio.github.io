Ejercicios Prácticos

Problemas resueltos y por resolver sobre equilibrio químico y principio de Le Chatelier

Ejercicio 1: Cálculo de Kc

Para la reacción N₂(g) + 3H₂(g) ⇌ 2NH₃(g), en equilibrio se encuentran:

  • [N₂] = 0.5 M
  • [H₂] = 1.5 M
  • [NH₃] = 0.8 M

Calcula el valor de Kc para esta reacción.

Ejercicio 2: Le Chatelier (Concentración)

Para el equilibrio: 2SO₂(g) + O₂(g) ⇌ 2SO₃(g) (ΔH = -197 kJ/mol)

Predice el efecto de:

  1. Aumentar [SO₂]
  2. Disminuir [SO₃]
  3. Añadir un gas inerte

Ejercicio 3: Sistema SIS

Para el sistema estireno-isopreno-estireno (SIS):

  1. Escribe la ecuación de equilibrio
  2. Calcula Kc si [S] = 0.2 M, [I] = 0.3 M, [SIS] = 0.15 M
  3. Analiza el efecto de aumentar temperatura (ΔH > 0)

Teoría del Equilibrio Químico

Conceptos fundamentales y el principio de Le Chatelier

Definición de Equilibrio Químico

Estado dinámico donde las velocidades de las reacciones directa e inversa son iguales, manteniendo constantes las concentraciones de reactivos y productos.

Características:

  • Se alcanza en sistemas cerrados a temperatura constante
  • Es dinámico (las reacciones no se detienen)
  • Las concentraciones permanecen constantes
  • Puede ser homogéneo o heterogéneo

Constante de Equilibrio (Kc)

Para una reacción general:

aA + bB ⇇ cC + dD

La expresión de Kc es:

Kc = [C]ᶜ[D]ᵈ / [A]ᵃ[B]ᵇ

Donde:

  • Corchetes [] representan concentraciones en equilibrio (mol/L)
  • Exponentes son los coeficientes estequiométricos
  • Sólidos y líquidos puros no se incluyen

Principio de Le Chatelier

"Si un sistema en equilibrio es perturbado, el sistema evolucionará para contrarrestar dicha perturbación y alcanzar un nuevo equilibrio."

Factores que afectan el equilibrio:

  1. Concentración: Aumentar [reactivos] desplaza hacia productos
  2. Presión: Aumentar P desplaza hacia menor nº de moles gaseosos
  3. Temperatura: Aumentar T favorece reacción endotérmica
  4. Catalizador: No afecta la posición de equilibrio
